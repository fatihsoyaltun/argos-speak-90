# Assessment Rubric

## Purpose

This document explains how speaking and review output should be evaluated during the Argos Speak 90 pilot.

This version intentionally uses simple, transparent self-check and admin review. It does not pretend to provide automatic AI scoring.

## What The App Checks

The app can check exact-answer review items such as:

- fill-blank answers
- short recall answers
- expected model sentences

These checks are useful for quick active recall, but they are not a full assessment of spoken English.

## What The App Does Not Score Automatically

Production-style review items are not automatically AI-scored in this version.

Examples:

- write one personal sentence
- add one reason
- ask one clarification question
- rewrite the idea in your own words
- summarize and add a next step

The app shows a model answer or correction help, but the learner should self-check the answer using the rubric below.

## 1-3 Self-Rating Rubric

Use this quick scale after Speak and production-style Review tasks:

### 1 - Needs another try

- I mostly copied the target line.
- I did not add a personal detail.
- My answer feels unfinished.
- My second try is not clearly better than my first try.

### 2 - Good enough for today

- I added at least one personal detail.
- I used one useful word, chunk, reason, or example.
- I finished the thought.
- My second try is a little clearer than my first try.

### 3 - Strong pilot answer

- I used my own context, not only the target line.
- I added a reason, example, clarification, or next step.
- My answer would make sense in a real everyday/work conversation.
- My second try is clearly more organized, clearer, or more natural.

## Quick Checklist

Before marking a day as strong, ask:

- Did I add a personal detail?
- Did I give a reason or example?
- Did I avoid copying the target line exactly?
- Is my second try clearly better than my first try?
- Did I finish the thought?

## Admin Review During Pilot

If the learner signs in and uses cloud sync, the admin can review synced written answers during the pilot.

Admin review should focus on learning progress, not punishment:

- Is the learner producing real output?
- Are answers becoming longer or clearer over time?
- Does the learner add personal details?
- Does the second try improve the first try?
- Are weak points recorded in Journal?

## Honest B1 Framing

Argos Speak 90 should be described as early B1-oriented speaking practice and a path toward functional early B1 in familiar daily/work contexts.

It does not guarantee full B1 proficiency, pronunciation accuracy, grammar accuracy, or real-time conversational fluency without pilot validation and human/AI feedback in a later phase.
