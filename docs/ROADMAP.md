# Argos Speak 90 — Roadmap

## Phase 1 — Core Skeleton
Goal:
Build the app shell and main pages.

Scope:
- Today
- Listen
- Words
- Speak
- Review
- Stats
- Settings
- mobile-first layout
- bottom navigation
- local-first structure

Output:
A clean, working app skeleton.

---

## Phase 2 — Listening Rebuild
Goal:
Make listening useful inside the app.

Scope:
- transcript-based listening drill
- key lines
- mini listening task
- output prompt
- optional browser speech playback
- no dependency on external links

Output:
A listening page that teaches, not just redirects.

---

## Phase 3 — Words Upgrade
Goal:
Make vocabulary more usable.

Scope:
- word cards
- pronunciation
- short Turkish meaning
- example sentence
- writing practice block
- small recall support

Output:
A stronger word-learning and usage experience.

---

## Phase 4 — Speak Engine
Goal:
Make speaking practice feel structured and real.

Scope:
- first try
- second try better
- self-check
- model answer preview later
- clearer completion logic

Output:
A more useful speaking workflow.

---

## Phase 5 — Review Engine
Goal:
Turn review into real active recall.

Scope:
- fill-in-the-blank
- short answer checks
- correct / try again feedback
- completion feel

Output:
A usable review system.

A real short learning track.
